SOMA SIMPLES

Já se tem o conhecimento de que aprender programação abre caminho para inúmeras possibilidades como desenvolvimento de aplicativos, websites e jogos. Tendo isso em vista, criamos o recurso Aprendendo++.

Aprendendo++ é um site voltado para o aprendizado dos princípios básicos da lógica de programação e algoritmo. O objetivo é ensinar os conceitos fundamentais e apresentar, por meio de exemplos e interações simples, uma linguagem de programação, a fim de que qualquer tipo de público seja capaz de começar a programar.

Lógica de Programação

É a capacidade que todo analista de sistemas precisa ter para resolver os problemas. A capacidade de dividir o problema em partes menores é uma etapa essencial e precisa ser levada em consideração quando nos deparamos com qualquer exercício/desafio. É nesse ponto que entra o conceito de algoritmo, descrito, geralmente, como uma sequência lógica de ações capaz de resolver um problema.
Porém esse conceito de algoritmo vai muito além. Uma receita de bolo, uma instrução escrita passo a passo por exemplo, é um exemplo simples de algoritmo. Segue um exemplo abaixo.

4 - Mascar a Bala

ALGORITMO SOMA
DECLARE A, B, SOMA: INTEIRO
INICIO
    LEIA (A)
    LEIA (B)
    SOMA ← A + B
    ESCREVA ("A soma é: " + SOMA)
FIM

#include < bits/stdc++.h>

using namespace std;
int main(){
    int A, B, SOMA;
    scanf("%d", &A);
    scanf("%d", &B);
    SOMA=A+B;
    printf("SOMA = %d\n", SOMA);
    return 0;
}
